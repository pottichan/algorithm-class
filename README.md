# 프로그래밍 과제 1: 반복/재귀로 n! 계산기 만들기

## 소개
이 프로그램은 정수 n(0 이상)에 대해 n! 값을 반복문(iterative)과 재귀(recursive) 방식으로 각각 계산하고, 실행 시간을 비교할 수 있습니다.

---

##  실행 방법
1. **Python 3**이 설치된 환경과 vscode 준비한다.
2. 소스코드 파일(`factorial_app.py`)을 다운로드한다.
3. 터미널(또는 명령 프롬프트)을 열고, 해당 파일이 있는 디렉토리로 이동한다.
4. 실행버튼을 눌러 프로그램을 실행한다.
